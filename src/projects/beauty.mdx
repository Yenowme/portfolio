beauty-saurus는 정적 웹 사이트 생성기 docusaurus를 기반으로, 비개발자도 코드 레벨이 아닌 그래픽 기반 인터페이스로 정적 웹 사이트를 쉽게 만들 수 있게 도와주는 오픈소스 프로젝트입니다.

사이트의 랜딩페이지 스타일링 외에도 드래그 앤 드롭으로 마크다운 파일을 추가해서 페이지를 생성하는 기능도 구현했습니다.
